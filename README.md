## 项目介绍
基于SSM框架的在线商城系统，系统分为用户和管理员两个角色，主要功能如下
用户：商品浏览、商品购买、购物车、商品下单、支付、我的订单、登录注册等
管理员：商品上架、商品分类、商品信息维护、发货、订单管理、用户管理等

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、BootStrap、jQuery
- 后端技术：Spring、SpringMVC、MyBatis

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL5.7及以上
